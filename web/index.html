Setup in minutes

1

Install Ollama

Download and install Ollama to provide the local AI runtime for DeepSeek Extension.

https://ollama.com
2

Pull DeepSeek R1

Run this command in your terminal to download the DeepSeek R1 model.

ollama run deepseek-r1
3

Install Extension

Install from VS Code Marketplace and start coding with AI assistance.

code --install-extension aryansrao.deekseek-extension
